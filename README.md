# Embedded-Sensor-Projects
A collection of embedded systems and sensor interfacing projects using Arduino and various electronic sensors.

This repository contains my embedded systems and sensor interfacing projects developed as part of my Electronics and Communication Engineering studies. The projects involve Arduino, STM32, ESP8266, various sensors, and IoT concepts.

## Technologies Used
- Embedded C
- Arduino IDE
- Sensors and Actuators
- IoT

## Projects
- Temperature Sensor Interfacing
- LDR Sensor
- Ultrasonic Sensor
- IR Sensor
- RFID Projects
