# Embedded Systems Projects

A collection of embedded systems and sensor interfacing projects developed as part of hands-on learning and experimentation in Electronics and Communication Engineering.

This repository contains basic projects involving various sensors, microcontrollers, and embedded programming concepts. The projects were created during collaborative learning sessions with fellow students, where we explored sensor interfacing, circuit design, and embedded software development together.

## Technologies Used

* Embedded C
* Arduino IDE
* Electronics and Sensor Modules
* IoT Fundamentals

## Topics Covered

* Temperature Sensors
* LDR Sensors
* Ultrasonic Sensors
* IR Sensors
* RFID Modules
* Serial Communication
* Sensor Data Acquisition
* Basic Embedded System Design

## Purpose

The goal of this repository is to document my learning journey in embedded systems, microcontrollers, and sensor interfacing through practical experiments and mini-projects.

## Note

These projects were developed as part of collaborative learning and experimentation with peers. They are intended for educational purposes and to showcase practical embedded systems skills.
